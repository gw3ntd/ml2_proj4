# Directory Description
- `compositions/`: folder containing .mp3 files of 6 generated compositions from 2 different models
- `jsb_chorales/`: folder containing the training, validation, and testing datasets
- `models/`: folder containing each trained model
- `pictures/`: folder containing learning curve graphics for each model
- `bach.ipynb`: notebook containing all of the preprocessing, model building, model training, model testing, and chorale generation
- `tuning_results.md`: results from hyperparameter tuning
